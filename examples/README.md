# Basic Usage Example

This example demonstrates the basic usage of vectra-crud library.

## Files

- `user-schema.ts` - Schema definition for user CRUD
- `user-datasource.ts` - Data source implementation (stub)
- `UserPage.vue` - Complete user management page

## Usage

```typescript
import { CrudProvider, DictProvider, CrudPage } from 'vectra-crud'
import { userSchema, userDataSource } from './user-schema'

// In your Vue component
```

See the example files for complete implementation.
