# vectra-crud Examples

This directory contains example usage of the vectra-crud library.

## Basic Usage Example

See `basic-usage.md` for a simple example of how to use the library.

## Running Examples

To try out these examples:

1. Install dependencies in your Vue 3 project
2. Install vectra-crud, element-plus, and vxe-table
3. Copy the example code into your Vue component
4. Import the necessary styles

```bash
pnpm add vectra-crud element-plus vxe-table
```

```javascript
import 'element-plus/dist/index.css'
import 'vxe-table/lib/style.css'
import 'vectra-crud/style.css'
```
